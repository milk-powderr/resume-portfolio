  • About
  • 'Slap' is a brand of citrus-flavored energy drinks. This project is a 4-pack that contains four different flavors: lemon, key lime, mandarin, and grapefruit.
  • Process
  • The name 'Slap' is attributed to bite of sourness from citrus fruits that is akin to getting slapped in the face. The package has a tab lock top and a crash lock bottom.
